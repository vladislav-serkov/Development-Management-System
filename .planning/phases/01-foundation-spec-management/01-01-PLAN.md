       - `status: Mapped[str] = mapped_column(String(20), nullable=False, default="pending")` — values: pending | processing | extracting | done | error | partial
       - `error_message: Mapped[Optional[str]] = mapped_column(Text, nullable=True)` — error details when status=error
       - `feature_count: Mapped[int] = mapped_column(Integer, default=0)` — detected feature count for quick access
       - `features: Mapped[list["Feature"]] = relationship("Feature", back_populates="document", lazy="selectin", cascade="all, delete-orphan")` — lazy="selectin" prevents MissingGreenlet errors in async context; cascade handles cleanup
       - Add `__repr__` returning `f"<Document id={self.id} filename={self.filename} status={self.status}>"`

       **Feature model (`__tablename__ = "features"`):**
       - `id: Mapped[int] = mapped_column(primary_key=True)`
       - `document_id: Mapped[int] = mapped_column(ForeignKey("documents.id", ondelete="CASCADE"), nullable=False, index=True)` — indexed FK with cascade delete
       - `name: Mapped[str] = mapped_column(String(255), nullable=False)`
       - `type: Mapped[str] = mapped_column(String(50), nullable=False)` — kafka_consumer | rest_endpoint | scheduled_task | unknown
       - `confidence: Mapped[float] = mapped_column(Float, nullable=False)` — 0.0-1.0

       Add a helper function at module level:
       ```python
       def feature_to_response(feature) -> FeatureResponse:
           """Convert Feature ORM to FeatureResponse, parsing business_logic JSON."""
           import json
           bl = None
           if feature.business_logic:
               try:
                   bl = json.loads(feature.business_logic)
               except json.JSONDecodeError:
                   bl = {"_raw": feature.business_logic}
           return FeatureResponse(
               id=feature.id,
               name=feature.name,
               type=feature.type,
               confidence=feature.confidence,
               summary=feature.summary,
               status=feature.status,
               business_logic=bl,
           )
       ```
