    1. Create `app/models/document.py` with SQLAlchemy ORM models:

       **Document model:**
       - `id: Mapped[int]` — primary key, autoincrement
       - `filename: Mapped[str]` — original PDF filename, not nullable
       - `uploaded_at: Mapped[datetime]` — default `datetime.utcnow`
       - `status: Mapped[str]` — default `"pending"`, values: pending | processing | done | error
       - `error_message: Mapped[Optional[str]]` — nullable, stores error details if status=error
       - `features: Mapped[list["Feature"]]` — relationship back_populates="document", lazy="selectin"
       - `__tablename__ = "documents"`

       **Feature model:**
       - `id: Mapped[int]` — primary key, autoincrement
       - `document_id: Mapped[int]` — ForeignKey("documents.id"), not nullable
       - `name: Mapped[str]` — feature name from Claude (e.g. "product-schedule-consumer"), not nullable
       - `type: Mapped[str]` — feature type: kafka_consumer | rest_endpoint | scheduled_task | unknown, not nullable
       - `confidence: Mapped[float]` — 0.0-1.0, not nullable
       - `summary: Mapped[Optional[str]]` — one-line description
       - `dependencies_json: Mapped[Optional[str]]` — JSON array of dependency names stored as TEXT
       - `business_logic: Mapped[Optional[str]]` — full JSON blob from 2nd Claude call stored as TEXT
       - `extracted_at: Mapped[datetime]` — default `datetime.utcnow`
       - `document: Mapped["Document"]` — relationship back_populates="features"
       - `__tablename__ = "features"`

       Use SQLAlchemy 2.0 Mapped[] annotation style (not legacy Column style).
       Import Base from app.database.

    2. Create `app/schemas/extraction.py` with Pydantic v2 schemas:

       **FeatureType(str, Enum):**
       - `kafka_consumer = "kafka_consumer"`
       - `rest_endpoint = "rest_endpoint"`
       - `scheduled_task = "scheduled_task"`
       - `unknown = "unknown"`

       **DetectedFeature(BaseModel):**
       - `name: str` — feature name extracted from PDF
       - `type: FeatureType` — classified type
       - `confidence: float` — 0.0 to 1.0, use `Field(ge=0.0, le=1.0)`
       - `summary: str` — one-line description
       - `dependencies: list[str]` — mentioned external services, tables, cache keys

       **FeatureDetectionResult(BaseModel):**
       - `features: list[DetectedFeature]`
       This is the model passed to `client.messages.parse(output_format=FeatureDetectionResult)`.

       **DocumentResponse(BaseModel):**
       - `id: int`
       - `filename: str`
       - `status: str`
       - `features: list[FeatureResponse]`
       - `uploaded_at: datetime`

       **FeatureResponse(BaseModel):**
       - `id: int`
       - `name: str`
       - `type: str`
       - `confidence: float`
       - `summary: str | None`
       - `business_logic: dict | None` — parsed from JSON text
       Configure `model_config = ConfigDict(from_attributes=True)` on response models.
