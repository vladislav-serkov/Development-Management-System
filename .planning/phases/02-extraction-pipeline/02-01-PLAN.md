3. Implement `_run_dedup_and_gaps` in `app/services/extraction.py`:

```python
async def _run_dedup_and_gaps(
    features: list[tuple[str, dict]],  # [(feature_name, business_logic_dict), ...]
    client: anthropic.AsyncAnthropic,
    model: str,
) -> DeduplicationResult:
```

- Build `context_blob = json.dumps({name: bl for name, bl in features}, ensure_ascii=False, indent=2)`
- Create message with 2 content blocks: (a) text block with context_blob + `cache_control: {"type": "ephemeral"}`, (b) text block with DEDUP_GAPS_PROMPT
- DEDUP_GAPS_PROMPT instructs Claude to analyze all features and return JSON with keys: "dependencies" (dict with "db", "external_api", "cache" lists), "overviews" (dict feature_name -> markdown), "gaps" (list of gap objects)
- Each dependency entry must have: name, type, used_by_features[], known_operations, and schema fields (columns for db, endpoints for api, structure for cache)
- Each gap must have: category, name, affected_features, what_missing, priority, suggestion (nullable)
- Each overview must be markdown with: feature type, summary, dependency links, business logic summary, gap references
- Parse response with `_extract_json_from_text()`, validate with `DeduplicationResult.model_validate()`
- Log cache stats with `_log_cache_stats(response.usage, "dedup_and_gaps")`
- Set max_tokens=8192

4. Implement `_store_dedup_results` in `app/services/extraction.py`:

async def _store_dedup_results(

    doc: Document,

) -> None:

- For each registry_type ("db", "external_api", "cache") in result.dependencies:
  - Validate registry_type is one of the known 3; log warning and skip unknown types
  - For each entry in the list, create DependencyEntry(document_id=doc.id, registry_type=registry_type, name=entry["name"], data_json=json.dumps(entry, ensure_ascii=False))
- For each gap in result.gaps, create GapEntry(document_id=doc.id, category=gap.category, name=gap.name, affected_features=json.dumps(gap.affected_features), what_missing=gap.what_missing, priority=gap.priority, suggestion_json=json.dumps(gap.suggestion) if gap.suggestion else None)
- For each feature_name in feature_orm_map:
  - If feature_name in result.overviews: set feature_orm.overview_md = result.overviews[feature_name]
  - Else (fallback): set feature_orm.overview_md = f"## {feature_name}\n\n{feature_orm.summary or 'No overview available.'}"
