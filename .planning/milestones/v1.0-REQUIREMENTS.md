## v1 Requirements — All Complete

### PDF Processing
- [x] **PDF-01**: Upload PDF via web UI (drag-and-drop + file picker) — Phase 1, 3

- [x] **PDF-04**: Support multi-feature PDFs (one PDF = multiple features) — Phase 1

- [x] **EXTR-03**: Extract external APIs to shared registry (no cross-feature duplicates) — Phase 2
